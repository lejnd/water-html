## 目录结构

~~~
-- css: less自动编译成 css 文件，不用手写
-- html: 所有静态HTML
-- images: 图片
-- less: 需要手写的less
-- vender: 第三方引用
-- gulpfile.js: 自动化工具gulp配置，用于编译less等
~~~

## 命令

~~~
首先全局安装gulp
npm install gulp -g

gulp 执行编译

gulp watch 执行自动编译（保存就编译）
~~~
